# Documentation Intelligence & Engineering Reports Guide

RepoLens generates and maintains code onboarding guides, database ER structures, API routers references, and developer manuals dynamically.

## 1. Document Generator Templates
We provide structured document generation planners:
* **README improvements**: Scans workspaces folders to build technologies listings and development setups steps.
* **Architecture manuals**: Renders high-level package components maps with visual Mermaid flowchart trees.
* **API routers listings**: Lists methods decorators, route paths, controller handles, and mapped ORM schemas.
* **Database ER layouts**: Maps entities constraints and indices with Mermaid schema charts.

---

## 2. Engineering Telemetry Reports
Reports run codebase evaluations:
* **Coupling Ratio**: Evaluates file import dependencies ratios.
* **Complexity rating**: Lists circular references and risk indices.
* **Health scores**: Combines documentation coverage metrics.
